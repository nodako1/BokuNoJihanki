# 素材来歴管理

## M0

| 種別 | 素材 | 作成方法 | 第三者素材 |
|---|---|---|---|
| 背景 | 夏の町・住宅・ゲームショップ・自販機・山 | Phaser Graphicsによるオリジナル手続き描画 | なし |
| アイコン | 自販機アイコン | プロジェクト内で作成したSVG | なし |
| BGM | 短いペンタトニック旋律 | Web Audio APIで実行時合成 | なし |
| 環境音 | セミを想起させるノイズ | Web Audio APIで実行時合成 | なし |
| 効果音 | 開始・ボタンクリック | Web Audio APIで実行時合成 | なし |

## 今後の記録項目

生成画像・音源を追加するたび、以下を追記する。

- ファイルパス
- 作成日
- 生成または録音方法
- 指示文または生成スクリプト
- 参照素材
- 人手による加工
- 権利確認
- SHA-256
