# アートディレクション

## 目標

日本の2000年代前後の夏休みを想起させる、ドット絵ではない高解像度2.5Dイラストを採用します。承認済みM1完成資料の中央ゲーム画面を、密度・暖かさ・奥行き・生活感の基準とします。ただし資料画像自体はゲームへ貼り付けず、実際に歩ける背景・小物・前景へ分解して再構築します。

## 固定ルール

- 横画面16:9、固定の見下ろし2.5Dカメラ
- 接地点を基準にしたYソート
- 左上からの光、暖色のハイライト、青緑の影
- 柔らかな輪郭と、細かな紙・葉・アスファルトのテクスチャ
- 実在ブランド、既存ゲーム、第三者キャラクターを使用しない
- 文字は原則ゲーム側で描画する
- 住宅街、公園、今後の駅・商店街・神社・山・学校・海も同じ光方向と色設計を継承する

## M1.1レイヤー

1. チャンク背景: 草地、遠景、道路、歩道、園路、地面テクスチャ
2. 固定オブジェクト: 住宅、樹木、植え込み、電柱、柵、公園設備
3. 生活小物: 郵便受け、自転車、道路反射鏡、自販機、花壇、ごみ箱
4. 動的主体: 主人公
5. 前景: 画面端の樹木・植栽など、主人公より前へ回り込む要素
6. 光と影: 接地影、雲影、窓明かり、街灯、自販機照明、時間帯グレーディング

## M1.1オリジナル素材

- 主人公: 上下左右×待機・歩行。左右は向きに対応した横顔
- 住宅: 4種類
- 樹木: 3種類
- 植栽: 生垣2種類、低木、花壇
- 道路設備: 電柱、街灯、道路反射鏡、郵便受け、自転車
- 公園設備: 看板、入口ゲート、ベンチ、滑り台、ブランコ、砂場、ごみ箱
- 商業小物: 架空デザインの自販機
- 背景: 住宅街西・東、公園西・東の4チャンク

素材は`src/game/world/m11*.ts`へ分割して管理します。アセット一覧と生成方法は`docs/ASSET_PROVENANCE.md`へ記録します。

## 時間帯

同じ場所が時刻によって変化することを重視します。

- 朝: 黄緑と淡い金色、長い柔らかな光
- 昼: 高彩度の緑、明瞭な道路・建物
- 夕方: 橙色の光、紫寄りの影、窓明かりの開始
- 夜: 青紺の色調、窓・街灯・自販機の発光、公園の微細な光粒

## ビジュアル完了判定

素材数ではなく、Browser Smokeが取得した実ゲーム画面で判断します。住宅街、公園、朝・昼・夕方・夜を承認済み資料と比較し、空白の多さ、背景密度、主人公の馴染み、奥行き、時間帯差を確認します。

M1.1は資料の画風を完全複製するものではなく、プロジェクト独自のテクスチャ付きSVG表現として再設計しています。将来の本番アート統一工程では、同じ構図とレイヤー構造を維持したまま、必要に応じてラスターペイント素材へ段階的に置き換えられます。
