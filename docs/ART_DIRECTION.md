# アートディレクション

## M1.2基準

ユーザー承認済みの「住宅街と公園が交差する夏の町」画像を、世界観・カメラ・画材・密度の最優先基準とする。

- 日本の2000年代前後の夏休み
- 斜め上からの固定2.5Dカメラ
- 手描きラスターペイントの瓦、壁、路面、葉、木漏れ日
- 暖色の直射光と青緑の影
- 道路、電線、住宅、植栽、生活小物が一枚の世界として馴染む密度
- 主人公はスマホでも読めるシルエットを保ちつつ背景と同じ輪郭・色温度へ合わせる

## 実装方式

M1.1のコード生成SVGはフォールバックと設計資料として残す。M1.2の表示は、承認済みマスターから生成する高解像度WebP背景、時間帯差分、透過前景、ラスタープレイヤーを使用する。

コンセプト画像をUIごと静止画として表示しない。資料用HUDと仮プレイヤーを除去し、背景・前景・実プレイヤー・衝突・インタラクションを分離する。追加エリアは同じカメラと光方向で別マスターを制作し、単純な鏡像だけで正式版全域を構成しない。

## 品質確認

モックアップではなくBrowser Smokeが取得した実画面を基準画像と比較する。朝・昼・夕方・夜、住宅街、公園、主人公、前景オクルージョン、境界継ぎ目を確認する。
