# M1 町を歩く基盤 仕様書

## 目的

住宅街と公園をロード無しで歩き、スマホ横画面で移動、衝突、奥行き、時間帯、音を確認できる状態を作る。自販機探索と経済はM2へ分離する。

## 完了条件

- 4方向移動、歩行差分、仮想スティック、WASD／矢印
- 追従カメラ、マップ外防止、入力消失時停止
- 住宅街2チャンク、公園2チャンク
- 現在・隣接・進行方向先読み、遠方解放
- 建物、木、植え込み、電柱、街灯、柵、ベンチ、遊具、自販機の衝突
- 接地点Yソート
- M0の時間帯と実マップ、光、環境音の同期
- safe-area、縦向き案内、開発HUD
- オリジナルSVGとプロシージャル音声
- CI、main、Vercel Production、文書更新

## 非対象

所持金、自販機調査、抽選、行動時間、帰宅、日付、NPC、会話、イベント、ショップ、インベントリ、セーブ、エンディング。

## マップ

ワールドは5120×720。道路の通行帯は主にY=559〜694、公園の園路はY=552〜694。プレイヤーの可動範囲はY=356〜690。右へ進むほど住宅街から公園へ移る。

## 品質

`npm run check`を必須とし、ProductionではスマホとPCの両方で操作、シームレス移動、衝突、時間帯、音、HUDを確認する。
