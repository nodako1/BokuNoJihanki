## マップ

ワールドは5120×720。道路の通行帯は主にY=559〜694、公園の園路はY=552〜694。プレイヤーの可動範囲はY=356〜690。右へ進むほど住宅街から公園へ移る。

内部チャンクは以下の4つです。

1. `residential-west`
2. `residential-east`
3. `park-west`
4. `park-east`

現在地、隣接、進行方向のチャンクを読み込み、十分に離れたチャンクを破棄します。

## 黒画面インシデント

生成SVGのpercent-encoded data URLをPhaserがbase64として復号し、`InvalidCharacterError`でScene初期化が停止しました。SVGをUTF-8バイト列からbase64へ変換する修正を行い、Browser Smokeを完了条件へ追加しました。
