# ぼくの自販機

**人生で一番バカで、一番楽しい夏休み。**

『ぼくの自販機』は、8月1日から8月31日まで夏の町を歩き回り、自販機の下やお釣り返却口を調べてお金を集め、ゲームショップで毎日3本だけ入荷するゲームソフトを買い集めるスマホ横画面向け探索RPGです。

自販機探索を軸に、お手伝い、都市伝説、プレゼント、プール、バス、埋蔵金探しなど、すべての遊びが「ゲームソフトを何本買えたか」という一つの評価につながります。

## 開発コマンド

```bash
npm install
npm run dev
npm run check
npm run build
npm run preview
```

Node.js 22を使用します。

## 開発方針

- `main`は常にVercelで確認可能な状態を維持する
- 実装は一時ブランチで行い、チェック成功後に`main`へマージする
- 恒久的な開発環境・ステージング環境は持たない
- 仕様、進捗、次の作業はリポジトリ内に保存する
- 画像、BGM、効果音はオリジナル素材のみを使用する
- 判断基準は「プレイヤーも登場人物も笑顔になれるか」

詳細は[`docs/`](./docs/)と[`PROJECT_STATE.json`](./PROJECT_STATE.json)を参照してください。
